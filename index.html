Built on Next.js native primitives

App Router, React Server Components, and Server Actions — with H3 hex discovery, a strict booking state machine, and privacy-preserving signals.

🗺️

H3 hex discovery

Listings and posts indexed at neighborhood resolution — surfaced via cached, cursor-paginated route handlers.

🔁

Request state machine

PLACED → DELIVERED → COMPLETED. Strict role-based transitions enforced inside typed Server Actions.

🛡️

Privacy-preserving signals

Reactions are HMAC-bucketed by hour so we get hot-spot data without storing user → listing edges.
